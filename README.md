# Israeli Football Database Project (SQL)

## Project Overview
This project involves the design and implementation of a relational database focused on top Israeli football players.
It demonstrates my skills in data modeling and SQL development.

## Key Features
* *Database Schema:* Structured tables for players, teams, and performance metrics.
* *SQL Queries:* Implementation of complex queries, including Joins and Data Manipulation.
* *Data Integrity:* Ensuring accurate and consistent data representation.

## Tools Used
* *SQL:* For database creation and querying.
* *Database Management System:* (e.g., MySQL / SQL Server).

## How to Use
1. Download the FirstDataBase - Naor Bitton.sql file.
2. Run the script in your SQL environment to generate the database and view the sample data.
